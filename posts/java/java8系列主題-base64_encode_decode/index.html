Java 8 - Base64 Encode and Decode

June 22, 2022

Base64 是一種以二進位制進行編碼的的表示方式。 Base64 以六個位元表示 64 個字元，包含 26 個拉丁字母大小寫、數字 0-9 以及兩個根據不同作業系統而不同的字元，編碼表格可以參考以下的兩個表格，第一個表格為 Base64 編碼表，第二個表格為 URL and Filename safe 的 Base64 編碼表，兩者的差異在於最後的兩個編碼字元。
來源：https://datatracker.ietf.org/doc/html/rfc4648
來源：https://datatracker.ietf.org/doc/html/rfc4648

Base64 的編碼轉換規則可以參考維基百科，維基百科所舉的例子是將 ASCII 編碼與 Base64 編碼進行轉換。

Base64 被應用在文字資料的場合，例如 Multipurpose Internet Mail Extensions (MIME) 或是郵件附件等，因此早在 Java 8 提供 Base64 Encode 及 Decode API 之前，就已經有其他 Base64 實作存在。

可以從 Base64 API 看出 JDK8 的 Base64 API 主要支援三中類型的 Base64 ：

  • Basic
  • URL and Filename safe
  • MIME

其中第一種和第二種 Base64 的差別在於前者編碼依據為上述的第一個表格，後者採用依據則是第二個表格。 MIME Base64 則時基於特定規則，像是編碼後每行最長不能超過 76 個字元。另外， Base64 裡提供的都是產生 Base.Encoder 以及 Base.Decoder 的工廠方法，從工廠方法取得實例後就可以對字串進行編碼及解碼。

Base64.Encoder base64Encoder = Base64.getEncoder();
Base64.Encoder base64MimeEncoder = Base64.getMimeEncoder();
Base64.Encoder base64UrlEncoder = Base64.getUrlEncoder();
Base64.Decoder base64Decoder = Base64.getDecoder();
Base64.Decoder base64MimeDecoder = Base64.getMimeDecoder();
Base64.Decoder base64UrlDecoder = Base64.getUrlDecoder();

在使用 Base.Encoder 及 Base.Decoder 進行編碼時，如果使用的是 String ，則事先需要先將 String 轉換為 byte[] 陣列在呼叫 encode() 或是 decode() 。

Class Base64.Encoder
  byte[] encode(byte[] src)
  
Class Base64.Decoder
  byte[] decode(byte[] src)

最後附上 Base64 API 進行編碼及解碼的範例：

Base64.Encoder base64Encoder = Base64.getEncoder();
Base64.Decoder base64Decoder = Base64.getDecoder();

String exampleString = "This is a test string with symbols _ and +."
String urlString = "https://docs.oracle.com/javase/8/docs/technotes/guides/lang/enhancements.html#jdk8";

System.out.println("Before encode: ");
System.out.println(exampleString);
System.out.println("After encode:");
byte[] encodedBytes1 = encoder.encode(exampleString.getBytes());
System.out.println(new String(encodedBytes1));
System.out.println("After decode:");
byte[] decodedBytes1 = decoder.decode(encodedBytes1);
System.out.println(new String(decodedBytes1));
System.out.println();

System.out.println("Before encode: ");
System.out.println(urlString);
System.out.println("After encode:");
byte[] encodedBytes2 = encoder.encode(urlString.getBytes());
System.out.println(new String(encodedBytes2));
System.out.println("After decode:");
byte[] decodedBytes2 = decoder.decode(encodedBytes2);
System.out.println(new String(decodedBytes2));
System.out.println();

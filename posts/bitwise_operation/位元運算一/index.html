位元運算（一）

August 15, 2022

位元運算是程式設計中對二進位數的操作，與一般加減及乘除運算相比，有運算速度更快、耗費資運更少的優勢，是優化程式非常有效的方式。

Java 提供位元運算子以及移位運算子：其中位元運算子包含 OR (|) 、AND (&) 、XOR（^）、NOT（~），移位運算子包含算術右移位運算子（>>>）、右移位運算子（>>）以及左移位運算子（<<）。

位元運算技巧

以下列出常見並且實用的位元運算，使用 1s 以及 0s 來表示全為 1 或是 0 的二進位序列：

  • x ^ 0s = x
  • x & 0s = 0
  • x | 0s = x
  • x ^ 1s = ~x
  • x & 1s = x
  • x | 1s = 1s
  • x ^ x = 0
  • x & x = x

二補數 (2’s complement)

二補數是一種用二進位表示有符號數的方法，電腦通常使用此方法表示正負數。正數的最高位元為 0 ，複數的最高位元為 1 ，正負數的轉換方式為，先將二進位表示數字做 NOT 運算再加 1 。以下是以四位元進行正負號轉換範例：

  1. 假設有一以二進位表示長度為四位元的數 0110 ，十進位表示為 +6
  2. 將此數的二進位序列進行 NOT 運算，得到 1001
  3. 將得到 NOT 運算的二進位序列再加 1 ，得到 1010 ，即為十進位表示的 -6
  4. 將 0110 與 1010 相加得到 10000 ，由於長度限制為四位元，因此得到 0000 ，即十進位表示的 0 ，驗證此轉換無誤

移位運算

移位運算包含算術移位運算與邏輯移位運算兩種類型，其中算術移位運算會保留原本數字的正負號，邏輯移位運算則否。

以下是六位元數 111001 進行移位運算範例：

  • 111001 >> 2 = 001110
  • 111001 >>> 2 = 100110
  • 111001 << 2 = 100100

值得注意的是 Java 僅提供邏輯左移位運算子（<<），但實際上其功用與算術左移位運算子（<<<）相同。
